# PairDiet

匿名の他人同士を性別・身長・体重が近い条件で1対1マッチングし、30日クールで一緒にダイエットするアプリ。核はサンクコスト（¥1,500チケット）＋社会的圧＋損失回避。健康アドバイスはしない（「続ける力」だけを担う）。

## 確定した重要決定
- 価格：初回¥1,500／更新¥2,000（一人あたり・3ヶ月）。「ふたりで¥1,500」は誤認NG。
- 割引：接近率→割引率をベース価格に適用（100%無料 / 80-99%50%off / 50-79%30%off / 20-49%10%off）。
- 配色：土台=藍紫#5b4bd6 ／ CTA=コーラル#ff5a3c ／ 良い結果・減量=緑#12b76a。
- オンボ：15画面。認証(Apple/Google)を最初＋18歳チェック。悩みクイズ→¥1,500で回収。体重/目標は体重計＋ドラム入力（目標はBMI18.5ガード）。健康免責の1枚あり。
- 年齢：18歳確認のみに使う（ダイエット計算には使わない）。
